方法¶

node.children()¶
Array<HTMLElement> children (selector [ , filter ])
获取符合选择器的所有非文字节点的子节点.
参数:
  • selector (string|HTMLCollection|Array<HTMLElement>) – 字符串格式参见 KISSY selector
  • filter (string|function) – 过滤条件, 格式参见 node.filter 的相应参数
返回:

获取符合选择器的所有非文字节点的子节点.
返回类型:

Array<HTMLElement>
