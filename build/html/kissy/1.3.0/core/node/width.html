NodeList.width()¶
NodeList width ()
得到当前节点列表第一个节点的计算宽度
返回类型:number

和 css('width') 的区别在于该函数返回不带单位的纯数值, 而前者则返回带单位的原始值(例如 400px ). 当需要数值计算时, 推荐该方法.

该方法也可以用来得到 windw 和 document 的宽度
KISSY.all(window).width(); // 得到浏览器可以区域的高度, 相当于 DOM.viewportWidth()
KISSY.all(document).width(); //得到 html 文档的高度, 相当于 DOM.docWidth()
NodeList width ( value )
设置当前列表每个元素的 css width 值.
参数:value (number|string) – 代表像素的整数值, 或数字加上其他单位的字符串值.

注解

在现代浏览器中, css width 属性不包括 padding , border 或者 margin.

例子可参考 height()
